Installation

To install the module into your local ArgoUML installation:

Retrieve the module from the website and extract put them in the ${ANT_HOME}/lib folder:

  • ant-diff-1.0-dev.jar
  • jrcs-1.0-dev.jar
  • commons-collections-3.1.jar
  • oro-2.0.8.jar

You can also use the provided batch file antdiff.bat to install the task in your local Ant installation by using the install option: 

				
D:\Workspaces\SoftwareEngineering\antdiff-1.0-dev>antdiff install
Buildfile: antdiff.xml

install:

install-lib:
      [get] Getting: http://ncelmaitre.nce.amadeus.net/maven/antdiff/jars/antdiff-1.0-dev.jar

install-lib:
      [get] Getting: http://ncelmaitre.nce.amadeus.net/maven/commons-collections/jars/commons-collections-3.1.jar

install-lib:
      [get] Getting: http://ncelmaitre.nce.amadeus.net/maven/jrcs/jars/jrcs-1.0-dev.jar

install-lib:
      [get] Getting: http://ncelmaitre.nce.amadeus.net/maven/oro/jars/oro-2.0.8.jar

BUILD SUCCESSFUL
Total time: 0 seconds

If you want to remove the libraries installed by this task use the uninstall option: 

				
D:\Workspaces\SoftwareEngineering\antdiff-1.0-dev>antdiff uninstall

Please notice that the installation with antdiff.bat isn't recommended, it's only an easy way to install the task.

In production environments, the installation of the needed libraries should be done manually, if possible in an other folder than ${ANT_HOME}/lib, and so the taskdef should reference the libraries in the appropriate folder.
