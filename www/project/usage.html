Usage

To use the task into your build files you should use it like this: 

				
<target name="test-one-diff" depends="init,taskdefs" 
    	description="--> Test the Ant Diff task">
  <echo>Compare file to file in Unified format</echo>
  <diff compare="${file1}" to="${file2}" type="unified" 
    output="patch.txt" error="errs.txt"/>    	
  <!-- folder to folder -->
  <echo>Compare dir to dir</echo>
  <diff compare="${dir1}" to="${dir2}"/>
  <echo>Compare dir to dir with inclusion filter</echo>
  <diff compare="${dir1}" to="${dir2}">
    <include name="*/*.txt"/>
  </diff>
</target>

It is also possible to use the batch file antdiff.bat , packaged in the distribution to install, remove and compare files.

To compare 2 files or folders with the batch file, simply give it the original file or folder as first argument and the revised file or folder as second argument: 

				
D:\Workspaces\SoftwareEngineering\antdiff-1.0-dev>antdiff src\test\toto1.txt src
